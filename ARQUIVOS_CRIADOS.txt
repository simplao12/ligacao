POST /api/devices/{code}    → Ativa device (refatorado)
DELETE /api/devices/{code}  → Cancela pareamento (refatorado)

TABELA: devices
  ├─ id              (PK)
  ├─ code            S + 7 alfanuméricos (UNIQUE)
  ├─ device_id       ANDROID_ID (UNIQUE)
  ├─ activated       true/false
  ├─ created_at      timestamp
  ├─ last_seen       timestamp
  └─ deleted_at      soft delete

TABELA: playlists
  ├─ id
  ├─ device_code     FK → devices.code
  ├─ kind            'xtream' | 'm3u'
  ├─ name
  ├─ host, username, password (Xtream)
  ├─ url, epg_url    (M3U)
  ├─ created_at
  └─ updated_at
